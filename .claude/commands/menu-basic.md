# สกิล: แพ็ก 1 — Basic เมนู QR เริ่มต้น

**ราคา: 990 บาท | ระยะเวลา: 2 วัน | แก้ไขฟรี: 1 ครั้ง**

สิ่งที่ลูกค้าได้รับ:
- เมนูออนไลน์สูงสุด 20 รายการ แยกหมวด + ใส่รูปอาหาร
- QR Code พร้อมใช้งาน
- ภาษาไทย
- แก้ไขฟรี 1 ครั้ง

---

## ขั้นตอนที่ 1 — รับข้อมูลจากลูกค้า

ถามลูกค้าให้ครบก่อนเริ่มงาน:

```
□ ชื่อร้าน
□ รายการเมนูสูงสุด 20 รายการ (ชื่อ + ราคา + หมวดหมู่)
□ รูปอาหารแต่ละเมนู (ถ้ามี)
□ โลโก้ร้าน (ถ้ามี)
□ สีหลักของร้าน (ถ้าไม่มีใช้สีแดง #8B2635 เป็น default)
```

## ขั้นตอนที่ 2 — ตั้งค่าโปรเจกต์

```bash
# clone โปรเจกต์ template
git clone https://github.com/[repo]/khaomoo-menu my-[ชื่อร้าน]-menu
cd my-[ชื่อร้าน]-menu
npm install
```

## ขั้นตอนที่ 3 — แก้ข้อมูลเมนู

เปิด `src/menuData.js` แล้ว:

1. **ลบเมนูตัวอย่างทั้งหมดออก** และใส่เมนูของลูกค้าแทน (ไม่เกิน 20 รายการ)
2. โครงสร้างแต่ละรายการ:
```js
{
  id: 1,
  cat: "<หมวดหมู่>",
  name: "<ชื่อเมนู>",
  price: <ราคา>,
  img: "<img_key>",
  badge: null,
  desc: "<คำอธิบายสั้นๆ>",
  opts: []   // Basic ไม่มีตัวเลือกเสริม
}
```
3. อัปเดต `CATS` array ให้ตรงกับหมวดหมู่ของร้านลูกค้า

## ขั้นตอนที่ 4 — ใส่รูปอาหาร

แปลงรูปเป็น base64 แล้วเพิ่มใน IMGS object ใน `src/App.jsx`:
```bash
# แปลงรูปเป็น base64
base64 -w 0 รูปภาพ.jpg
```
ใส่ผลลัพธ์ใน:
```js
const IMGS = {
  ชื่อ_key: "data:image/jpeg;base64,<base64 string>",
  ...
}
```

## ขั้นตอนที่ 5 — ปรับสีแบรนด์

ใน `src/App.jsx` หาบรรทัด `const C = {` แล้วแก้:
```js
const C = {
  red: "<สีหลักร้าน>",   // เช่น "#8B2635"
  cream: "#FAF7F2",
  dark: "#1A1A1A",
  gold: "#F0B030",
  border: "#EDE8DF",
  muted: "#B08060"
};
```

## ขั้นตอนที่ 6 — Build และตรวจสอบ

```bash
npm run build
# ต้องไม่มี error — เปิด dist/index.html ทดสอบ
```

## ขั้นตอนที่ 7 — Deploy และส่ง QR Code

```bash
# Deploy บน Netlify (ฟรี)
npx netlify-cli deploy --prod --dir=dist

# หรือ Vercel
npx vercel --prod
```

หลัง deploy ได้ URL → สร้าง QR Code ที่ [qr-code-generator.com](https://www.qr-code-generator.com) แล้วส่งให้ลูกค้า

## Checklist ก่อนส่งงาน

```
□ เมนูครบ ราคาถูกต้องทุกรายการ
□ รูปอาหารแสดงครบ ไม่มีรูปแตก
□ หมวดหมู่กรองได้ถูกต้อง
□ QR Code สแกนแล้วเปิดเมนูได้
□ ทดสอบบนมือถือ (iOS + Android)
□ ชื่อร้านแสดงถูกต้อง
```

## หมายเหตุ

- **ไม่รวม** Supabase / LINE OA / ตัวเลือกเมนู → อยู่ในแพ็ก 2 และ 3
- ถ้าลูกค้าอยากแก้ราคาเองในอนาคต → แนะนำ upgrade แพ็ก 2 (Standard)
- แก้ไขฟรี 1 ครั้ง — ครั้งที่ 2 เป็นต้นไปคิดเพิ่ม

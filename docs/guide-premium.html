📋 Checklist ข้อมูลที่ต้องส่ง

  • ชื่อร้าน (ไทย + อังกฤษ) เช่น "ข้าวมูหมู" และ "Khaomoo Restaurant"
  • รายการเมนูทั้งหมด ชื่อ + ราคา + หมวด + ตัวเลือก เช่น "ขนาด S/M/L" ส่งเป็น Excel หรือรูปถ่ายเมนูได้เลย
  • รูปอาหารทุกรายการ ยิ่งครบยิ่งดี ถ่ายจากมือถือแสงสว่างได้เลย
  • โลโก้ร้าน (PNG พื้นหลังใส หรือ JPG) โลโก้จะแสดงในหัวเมนู
  • สีหลัก + สีรองของแบรนด์ เช่น "แดงเลือดหมู + ครีม" หรือ hex code ก็ได้
  • LINE OA Channel Access Token ขอได้จาก LINE Developer Console → Messaging API → Issue token (ขอแบบ Long-lived ไม่หมดอายุ)
  • อีเมลสำหรับสร้างบัญชี Supabase ใช้ login เข้าแก้ราคาและดูออเดอร์ในภายหลัง
💡 LINE OA Token คืออะไร? คือรหัสลับที่ใช้ให้ระบบเมนูส่งข้อความเข้า LINE OA ร้านของคุณได้
ขอได้ที่ developers.line.biz → เลือก Channel → Messaging API → Issue a Longterm channel access token
ถ้าไม่แน่ใจ — ทักเราได้เลย เราจะแนะนำขั้นตอนทีละขั้น
⚠️ ห้ามส่ง Token ผ่าน Social Media ส่ง LINE OA Token ให้เราผ่าน LINE Direct (1:1) เท่านั้น อย่าโพสต์ลง Group หรือ Feed เพราะคนอื่นเอาไปใช้ได้

❓ คำถามที่พบบ่อย

ออเดอร์ไม่เข้า LINE OA ทำอย่างไร?
ตรวจสอบว่า LINE OA ยังออนไลน์ และ Token ไม่ถูกลบ ถ้ายังผิดปกติ — แจ้งเราทาง LINE เราตรวจสอบให้ภายใน 24 ชั่วโมง
ลูกค้าสแกน QR แล้วไม่เปิด?
ใช้กล้องมือถือปกติ (ไม่ต้องโหลดแอปพิเศษ) ถ้ายังไม่ขึ้น → ลองกดค้างที่ QR แล้วเลือก "เปิดลิงก์" หรือใช้กล้องใน LINE
Supabase ขึ้น "Paused" ทำอย่างไร?
เข้า supabase.com → คลิกโปรเจกต์ → กดปุ่ม Resume project สีเขียว → รอ 1–2 นาที → Refresh หน้าแอป (เกิดจาก Free tier หยุดอัตโนมัติเมื่อไม่ใช้ 7 วัน)
อยากเพิ่มตัวเลือกเมนูใหม่ เช่น ท็อปปิ้งใหม่ ทำเองได้ไหม?
ส่วนนี้ต้องให้เราทำให้ครับ แจ้งรายละเอียดมาได้เลย นับเป็นการแก้ไข 1 ครั้ง
อยากเพิ่มหมวดหมู่ใหม่ หรือเปลี่ยนชื่อหมวด ทำได้ไหม?
ได้ครับ ต้องให้เราทำให้ แจ้งชื่อหมวดใหม่มาได้เลย นับเป็นการแก้ไข 1 ครั้ง
LINE Token หมดอายุไหม?
ถ้าขอแบบ Long-lived token (ตามที่เราแนะนำ) — ไม่หมดอายุ ใช้ได้ตลอด ไม่ต้อง renew ทุกเดือน
ใช้สิทธิ์แก้ไขฟรีหมดแล้ว ครั้งต่อไปเสียเท่าไร?
แก้ข้อความ/ราคา → 150฿/ครั้ง · เพิ่มเมนูพร้อมรูป → 200฿/รายการ · เพิ่มตัวเลือก/หมวด → 300฿ · แก้ดีไซน์ → เริ่มต้น 500฿

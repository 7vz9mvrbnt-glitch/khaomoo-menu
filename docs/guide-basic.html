01
บทที่หนึ่ง
ยินดีต้อนรับ!
คุณได้อะไรบ้าง?
ทำความเข้าใจสิ่งที่คุณได้รับในแพ็ก Basic

ขอบคุณที่เลือกใช้บริการครับ ในแพ็กนี้คุณจะได้รับ เมนูอาหารออนไลน์ พร้อมใช้งาน ที่ลูกค้าสแกน QR Code แล้วดูเมนูได้ทันทีบนมือถือ โดยไม่ต้องดาวน์โหลดแอปใด ๆ

✅ สิ่งที่คุณได้รับในแพ็กนี้

  • 🍽️
    เมนูออนไลน์สูงสุด 20 รายการ พร้อมรูปภาพ แสดงรูปอาหาร ชื่อ ราคา และคำอธิบาย ดูสวยงามบนมือถือทุกรุ่น
  • 📂
    แยกหมวดหมู่ กรองเมนูได้ ลูกค้ากดเลือกหมวดได้ทันที เช่น "ของทานเล่น" "เครื่องดื่ม" หาเมนูเจอเร็วขึ้น
  • 📱
    QR Code ความละเอียดสูง พิมพ์ได้ทุกขนาด ไฟล์ PNG ขนาดใหญ่ พิมพ์บนกระดาษ สติกเกอร์ อคริลิค หรือกรอบได้เลย
  • 🔗
    ลิงก์เมนูออนไลน์ ส่งลิงก์ผ่าน LINE / Facebook ให้ลูกค้าเปิดดูได้โดยไม่ต้องสแกน QR
  • 🔧
    แก้ไขฟรี 1 ครั้ง (ภายใน 30 วัน) ขอแก้ชื่อเมนู ราคา หรือรูปภาพได้ 1 รอบ หลังจากนั้นคิดเพิ่มตามงาน
💚

"เมนูสวยงาม ดูมืออาชีพ
ลูกค้าดูรูปก่อนสั่ง
ตัดสินใจเร็วขึ้น ยอดขายดีขึ้น"

📌 เปรียบเทียบกับเมนูกระดาษ เมนูกระดาษพอราคาเปลี่ยน → ต้องพิมพ์ใหม่ทั้งเล่ม แต่เมนู QR Code → ส่งข้อมูลให้เราแก้ ราคาใหม่ขึ้นทันที ไม่ต้องพิมพ์ซ้ำ ประหยัดค่าพิมพ์ระยะยาว
02
บทที่สอง
ข้อมูลที่ต้องเตรียม
ส่งให้เรา
รวบรวมสิ่งเหล่านี้ก่อน แล้วส่งให้เราทาง LINE

ก่อนเราเริ่มทำงาน คุณต้องส่งข้อมูลเหล่านี้มาให้ก่อนนะครับ ยิ่งข้อมูลครบ งานยิ่งเสร็จเร็ว

📋 Checklist ข้อมูลที่ต้องส่ง

  • ชื่อร้าน ชื่อที่ต้องการแสดงในเมนู เช่น ร้านข้าวขาหมูต้นตำรับ
  • รายการเมนูทั้งหมด (สูงสุด 20 รายการ) ชื่อเมนู + ราคา + หมวดหมู่ ส่งเป็น Excel, Note, หรือถ่ายรูปเมนูเดิมมาได้เลย
  • รูปอาหาร (ถ้ามี) รูปละเอียด สว่าง พื้นหลังสะอาด ส่งเป็น JPG/PNG ถ่ายจากมือถือได้เลย
  • โลโก้ร้าน (ถ้ามี) ไฟล์ PNG หรือ JPG โลโก้หน้าร้าน หรือตราสินค้า
  • สีหลักของร้าน สีที่ต้องการใช้ เช่น "แดงเลือดหมู" "น้ำตาลโกโก้" หรือ code สี #XXXXXX ก็ได้
📸 เคล็ดลับถ่ายรูปอาหาร ถ่ายในที่แสงสว่าง ไม่ต้องมีกล้องแพง — มือถือก็ได้ผลดีมาก
ใส่ในจาน/ชาม ให้เห็นหน้าอาหาร อย่าถ่ายจากมุมต่ำเกินไป
พื้นหลังสีเรียบหรือโต๊ะไม้ดูดีมาก
⚠️ ยังไม่มีรูปก็ส่งได้ ถ้ายังไม่มีรูปอาหาร — ส่งข้อมูลอื่นมาก่อนได้เลยครับ เราจะใส่รูป placeholder ไว้ก่อน แล้วค่อยส่งรูปจริงมาทีหลัง (นับเป็น 1 ครั้งของสิทธิ์แก้ไขฟรี)
03
บทที่สาม
วิธีใช้ QR Code
ที่ร้าน
ตั้งแต่พิมพ์ QR Code ไปจนถึงลูกค้าสแกนดูเมนู

🖨️ ขั้นตอนฝั่งร้าน — ติดตั้ง QR Code

1
รับไฟล์ QR Code จากเราทาง LINE

เราส่งไฟล์ PNG ให้ทาง LINE พร้อมลิงก์เมนูด้วย

2
พิมพ์ QR Code ออกมา

พิมพ์บนกระดาษธรรมดา สติกเกอร์ หรืออคริลิคก็ได้ ขนาดแนะนำ 8×8 ซม. ขึ้นไป

3
ติดที่โต๊ะหรือเคาน์เตอร์

วางให้ลูกค้าเห็นชัดเจน ไม่อยู่ในที่มืด ใส่กรอบอคริลิคดูดีและทนทาน

✅ แนะนำติดทุกโต๊ะ ถ้ามีหลายโต๊ะ — พิมพ์หลายแผ่นได้เลย ใช้ QR Code ไฟล์เดิมทั้งหมด ลิงก์เดียวกัน

📱 ขั้นตอนฝั่งลูกค้า — สแกนดูเมนู

1
เปิดกล้องมือถือปกติ (ไม่ต้องโหลดแอปพิเศษ)

ทั้ง iPhone และ Android ใช้กล้องธรรมดาได้เลย

2
ชี้กล้องไปที่ QR Code บนโต๊ะ

กล้องจะจับ QR Code อัตโนมัติ ไม่ต้องกดอะไร

3
กดลิงก์ที่ขึ้นมาบนหน้าจอ

กด Open / เปิดลิงก์ → เมนูเปิดในเบราว์เซอร์ทันที

4
ดูเมนู เลือกรายการ แจ้งพนักงาน

ดูรูปอาหาร ราคา อ่านคำอธิบาย แล้วแจ้งพนักงานสั่ง

📱 สิ่งที่ลูกค้าเห็นเมื่อเปิดเมนู
🍖 ขาหมูพะโล้เตาถ่าน฿180
🍚 ข้าวขาหมูพะโล้เตาถ่าน฿60
🍵 กาแฟสด Premium฿65
…และอีก 17 รายการ
⚠️ ถ้าลูกค้าสแกนแล้วไม่ขึ้น? ลองกดค้างที่ QR Code → "เปิดลิงก์" หรือ ใช้แอป LINE สแกนก็ได้เช่นกัน
ตรวจสอบว่า QR Code ไม่เปื้อน ไม่ย่น และอยู่ในที่แสงสว่างพอ
04
บทที่สี่
คำถามที่พบบ่อย
ปัญหาที่มักเจอ และวิธีแก้ไขเบื้องต้น
ลูกค้าสแกน QR แล้วไม่เปิด?
ใช้กล้องมือถือปกติ (ไม่ต้องโหลดแอปพิเศษ) ถ้ายังไม่ขึ้น → ลองกดค้างที่ QR แล้วเลือก "เปิดลิงก์" หรือใช้กล้องใน LINE สแกนแทนได้
อยากแก้ราคาเมนู ทำได้ไหม?
ได้ครับ ส่งรายการที่ต้องการแก้ให้เราทาง LINE เราจะอัปเดตให้ภายใน 1 วันทำการ (นับเป็น 1 ครั้งของสิทธิ์แก้ไขฟรี)
ถ้าต้องการแก้ราคาเองได้ตลอดเวลา แนะนำอัปเกรดเป็น แพ็ก Standard
อยากเพิ่มเมนูใหม่ ทำอย่างไร?
ส่งชื่อเมนู ราคา และรูปภาพมาให้เรา — เราจะเพิ่มให้ (ถ้ายังมีสิทธิ์แก้ไขฟรีอยู่ จะใช้สิทธิ์นั้น ถ้าหมดแล้วคิดตามงาน)
เมนูขาดรูปภาพได้ไหม?
ได้ครับ ระบบจะแสดงรูป placeholder (รูปเปล่า) แทน แต่แนะนำให้ใส่รูปจริง เพราะลูกค้าเห็นรูปแล้วตัดสินใจสั่งได้ง่ายขึ้นมาก
ใช้สิทธิ์แก้ไขฟรีหมดแล้ว ครั้งต่อไปเสียเท่าไร?
แก้ข้อความ / ราคา → 150฿/ครั้ง · เพิ่มเมนูพร้อมรูป → 200฿/รายการ
ทักมาได้เลย เราจะ quote ราคาให้ก่อนเสมอ
ต้องการลิงก์เป็นชื่อร้านเอง เช่น menu.ชื่อร้าน.com ทำได้ไหม?
ได้ครับ ต้องซื้อ domain ก่อน (ประมาณ 500–800฿/ปี) แจ้งเราแล้วจะตั้งค่าให้ มีค่าบริการเพิ่มเติม ทักมาสอบถามราคาได้เลย
📞 ติดต่อเรา มีคำถามหรือปัญหาที่ไม่พบใน FAQ → ทักเราทาง LINE ได้เลย
ตอบกลับภายใน 24 ชั่วโมงในวันทำการ (จันทร์–เสาร์)

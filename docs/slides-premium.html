💡 ข้อเสนอแนะ

วิธีเร็วกว่า — ใช้ SQL แทน

🎯 ทำไม SQL ถึงง่ายกว่า Table Editor? Table Editor: ต้องหาแถว → เลื่อนหา column → คลิก → แก้
SQL: แค่พิมพ์ชื่อเมนู → กด Run → เสร็จ ใช้เวลาไม่ถึง 10 วินาที

เมนูซ้าย → SQL Editor → New Query → วางโค้ดนี้ → กด Run

-- 🙈 ซ่อนเมนู (เปลี่ยนแค่ชื่อเมนู) UPDATE public.menu SET is_available = false WHERE name ILIKE '%ชุดไส้%'; -- 👁️ เปิดกลับมา UPDATE public.menu SET is_available = true WHERE name ILIKE '%ชุดไส้%';
✏️ เปลี่ยนแค่คำในกรอบเหลือง ชุดไส้ = ชื่อ (หรือส่วนของชื่อ) เมนูที่ต้องการซ่อน
เช่น ถ้าต้องการซ่อน "ขาหมูพะโล้เตาถ่าน" ก็พิมพ์ ขาหมู
⚠️ ILIKE คืออะไร? คือค้นหาชื่อที่มีคำนั้นอยู่โดยไม่สนตัวเล็ก/ใหญ่ เช่น '%ไส้%' จับทุกเมนูที่มีคำว่า "ไส้" ในชื่อ

แก้ปัญหา

Supabase ขึ้น "Paused" — ทำอย่างไร?

⚠️ Paused เกิดจากอะไร? Supabase แบบฟรีจะหยุดทำงานอัตโนมัติถ้าไม่ได้ใช้งาน 7 วัน
เมนูลูกค้ายังดูได้ แต่การแก้ราคาจะยังไม่มีผล
  • 1
    เข้า supabase.com → Sign in → คลิกโปรเจกต์ จะเห็นสถานะ "Paused" หรือ "Unhealthy"
  • 2
    กดปุ่มสีเขียว "Resume project" โปรเจกต์จะเริ่มต้นใหม่ ใช้เวลา 1–2 นาที
  • 3
    รอ 2 นาที → กด Refresh หน้าแอปเมนู ทุกอย่างกลับมาปกติ ราคาอัปเดตได้ตามเดิม
💡 ป้องกัน Paused — เข้าสัปดาห์ละครั้ง แค่เปิดหน้า Supabase Table Editor ก็นับว่า active แล้ว ไม่ต้องทำอะไรเพิ่ม
📌 ถ้าร้านเปิดทุกวัน แนะนำ Supabase Pro $25/เดือน — ไม่มี pause อัตโนมัติ ไม่ต้องกังวลเรื่องนี้อีก

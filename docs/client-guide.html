ระบบเมนู QR Code คืออะไร?

ระบบเมนู QR Code คือ เมนูอาหารออนไลน์ ที่ลูกค้าสแกน QR Code แล้วดูเมนูและสั่งอาหารได้ทันทีบนมือถือ โดยไม่ต้องดาวน์โหลดแอปใด ๆ ทั้งสิ้น

🔄 ขั้นตอนการทำงาน

1
🖨️ พิมพ์ QR Code ติดที่โต๊ะ

รับไฟล์ QR Code จากเรา → พิมพ์ติดโต๊ะ / เคาน์เตอร์ / บอร์ดหน้าร้าน เป็นสติกเกอร์ กระดาษ หรืออคริลิค

2
📱 ลูกค้าสแกนด้วยกล้องมือถือ

เปิดกล้องปกติ → ชี้ที่ QR Code → กดลิงก์ที่ขึ้นมา → เมนูเปิดในเบราว์เซอร์ทันที ใช้ได้ทั้ง iPhone และ Android

3
🛒 เลือกเมนู เพิ่มลงตะกร้า กดสั่ง

ดูรูปอาหาร อ่านคำอธิบาย เลือกตัวเลือก (ขนาด/เนื้อ/ความหวาน) → เพิ่มลงตะกร้า → ยืนยัน

4
💬 ออเดอร์เข้าร้านทันที (แพ็ก Premium)

ร้านได้รับออเดอร์พร้อมรายการ ราคา หมายเลขโต๊ะ ผ่าน LINE OA โดยอัตโนมัติ

🌟 ข้อดีเทียบกับเมนูกระดาษแบบเดิม

  • 💰
    แก้ราคาได้ทันที ไม่ต้องพิมพ์ใหม่ เปลี่ยนราคาวันนี้ → ลูกค้าเห็นราคาใหม่วันนี้เลย ประหยัดค่าพิมพ์เมนูซ้ำ
  • 🚫
    ซ่อนเมนูที่หมดได้ชั่วคราว สัตว์เลี้ยงหมด หรือวัตถุดิบหมด → ซ่อนเมนูนั้น 1 คลิก โดยไม่ต้องลบออกถาวร
  • 🖼️
    ลูกค้าดูรูปอาหารก่อนสั่ง รูปสวยช่วยให้ลูกค้าตัดสินใจเร็วขึ้น และยอดออเดอร์ต่อโต๊ะมักสูงขึ้น
  • 🌐
    รองรับ 2 ภาษา ไทย / อังกฤษ (แพ็ก Standard & Premium) กดสลับได้ทันที ลูกค้าต่างชาติสั่งอาหารเองได้ ไม่ต้องรอพนักงานแปล
  • 📲
    ออเดอร์เข้า LINE OA ลดความผิดพลาด (แพ็ก Premium) รายการชัดเจน ครบถ้วน ไม่มีการได้ยินผิด หรือจดผิด
  • 🏠
    รองรับ Delivery สั่งส่งถึงบ้าน (แพ็ก Premium) ลูกค้ากรอกที่อยู่ + เลือกรอบส่ง → ร้านได้รับออเดอร์ครบในที

  • 🔗

📲 Flow การสั่งอาหาร (Dine In — ทานที่ร้าน)

1
ลูกค้าสแกน QR ที่โต๊ะ → เลือกโต๊ะ → เลือกเมนู

ดูรูป อ่านคำอธิบาย → กดเปิดรายการ → เลือกตัวเลือก (เนื้อ / ขนาด / ความหวาน / ท็อปปิ้ง)

2
ตรวจรายการในตะกร้า → กดยืนยัน

เลือกวิธีชำระเงิน: QR PromptPay หรือ เงินสด

3
ออเดอร์ส่งเข้า LINE OA ร้านทันที

ร้านได้รับ: โต๊ะ X · รายการเมนู · ราคารวม · วิธีชำระ ครบถ้วน ไม่มีการพิมพ์ผิด

4
ลูกค้าสแกน QR PromptPay จ่ายเงิน (ถ้าเลือกโอน)

แสดง QR PromptPay บนหน้าจอ → โอน → แนบสลิปส่ง LINE OA

🏠 Flow การสั่ง Delivery (ส่งถึงบ้าน)

2
กรอกข้อมูลจัดส่ง

ชื่อ · เบอร์โทร · ที่อยู่ · เลือกรอบจัดส่ง (เช้า / บ่าย)

3
ยืนยันออเดอร์ → ชำระผ่าน QR PromptPay

โอนเงิน → แนบสลิป + ส่งโลเคชั่น (pin บ้าน) ผ่าน LINE OA

4
ร้านได้รับออเดอร์ + ที่อยู่ครบใน LINE OA

วิธีแก้ราคา / เพิ่มเมนูด้วยตัวเอง

สำหรับแพ็ก Standard และ Premium · ไม่ต้องรู้โค้ด · ทำผ่านเบราว์เซอร์ปกติ

ℹ️ ทำได้ผ่านเว็บ Supabase (supabase.com) เราจะส่ง link + email + password ให้หลังส่งมอบงาน เข้า login ได้เลยทันที

🔖 แก้ราคาหรือชื่อเมนู

1
เปิดเบราว์เซอร์ → ไปที่ supabase.com → Sign in

กรอก email + password ที่เราส่งให้

2
เลือกโปรเจกต์ร้านของคุณ

จะเห็นชื่อโปรเจกต์ (ชื่อร้าน) ในหน้า Dashboard

3
เมนูซ้ายมือ → คลิก "Table Editor"

จะเห็นรายการตาราง → คลิกตาราง menu

4
คลิกเซลล์ที่ต้องการแก้ → พิมพ์ค่าใหม่ → กด Enter

แก้ได้ทั้ง price (ราคา), name (ชื่อ), description (คำอธิบาย)

✅ อัปเดตทันทีเลยครับ บันทึกแล้วเมนูในแอปอัปเดตภายใน 1–2 วินาที ลูกค้าโหลดหน้าใหม่จะเห็นราคาล่าสุดเลย

➕ เพิ่มเมนูใหม่

1
Table Editor → ตาราง menu → กดปุ่ม "Insert row"

หน้าต่าง "New row" เปิดขึ้น

2
กรอกข้อมูลเมนูใหม่

id (เลขไม่ซ้ำ) · cat (หมวดหมู่) · name (ชื่อเมนู) · price (ราคา) · description (คำอธิบาย)

3
กด Save → เมนูขึ้นในแอปทันที

หมายเหตุ: รูปภาพต้องให้เราอัปเดตให้ (นับเป็นการแก้ไข 1 ครั้ง)

🔧 แก้ด้วย SQL Editor (เร็วกว่า — สำหรับแก้หลายรายการพร้อมกัน)

ถ้าต้องการแก้ราคาหลายเมนูพร้อมกัน ใช้ SQL Editor เร็วกว่ากดทีละช่องมาก
ไปที่: เมนูซ้าย → SQL Editor → New query → วางโค้ดด้านล่าง → กด Run

-- แก้ราคาเมนูเดียว (id คือหมายเลขเมนู) UPDATE public.menu SET price = 80 WHERE id = 4; -- แก้ราคาหลายเมนูพร้อมกัน UPDATE public.menu SET price = 70 WHERE id IN (4, 5, 6); -- แก้ชื่อเมนู UPDATE public.menu SET name = 'ขาหมูพะโล้สูตรใหม่' WHERE id = 1;
⚠️ ถ้า Supabase ขึ้น "Paused" หรือ "Unhealthy" Supabase แบบฟรีจะหยุดทำงานอัตโนมัติเมื่อไม่มีการใช้งาน 1 สัปดาห์
วิธีแก้: เข้า supabase.com → คลิกโปรเจกต์ → กดปุ่ม Resume project → รอ 1–2 นาที → ใช้งานได้ปกติ

ถ้าร้านเปิดทุกวัน แนะนำอัปเกรด Supabase Pro ($25/เดือน) เพื่อไม่ให้ถูก pause

บทที่เจ็ด
วิธีซ่อน / แสดงเมนูชั่วคราว

ถ้าวัตถุดิบหมด หรืออยากปิดเมนูบางอย่างชั่วคราว — ทำได้ใน 3 คลิก โดยไม่ต้องลบเมนูออกถาวร

📌 ฟีเจอร์นี้มีใน Standard และ Premium เท่านั้น แพ็ก Basic ต้องให้เราแก้ให้

วิธีซ่อนเมนู (ปิดชั่วคราว)

1
Supabase → Table Editor → ตาราง menu

หาเมนูที่ต้องการซ่อน

2
มองหาคอลัมน์ "is_available"

ค่าปัจจุบัน = true (แสดง)

3
คลิกเซลล์ → เปลี่ยนเป็น false → กด Enter

เมนูนั้นหายจากแอปทันที ลูกค้าจะไม่เห็น

4
อยากแสดงอีกครั้ง → เปลี่ยน is_available กลับเป็น true

เมนูกลับมาแสดงทันที

⚡ วิธีด่วนผ่าน SQL Editor
-- ซ่อนเมนู (id = หมายเลขเมนูที่ต้องการ) UPDATE public.menu SET is_available = false WHERE id = 5; -- แสดงเมนูอีกครั้ง UPDATE public.menu SET is_available = true WHERE id = 5; -- ซ่อนหลายเมนูพร้อมกัน UPDATE public.menu SET is_available = false WHERE id IN (3, 5, 7);
✅ ไม่ต้องกลัวข้อมูลหาย การซ่อน (false) ไม่ได้ลบข้อมูลออก — แค่ซ่อนจากหน้าแอปเท่านั้น เปลี่ยนกลับเป็น true เมื่อไรก็ได้

บทที่แปด
คำถามที่พบบ่อย (FAQ)
ลูกค้าสแกน QR Code แล้วไม่เปิด?
ลองใช้กล้องมือถือปกติ (ไม่ต้องโหลดแอปพิเศษ) ถ้ายังไม่ขึ้น → ลองกด QR Code ใน Gallery แล้วกด "เปิดลิงก์" หรือใช้แอป LINE สแกนก็ได้เช่นกัน
แก้ราคาใน Supabase แล้ว แต่เมนูในแอปยังเป็นราคาเดิม?
โหลดหน้าใหม่ (ดึงหน้าจอลง / Refresh) ระบบจะดึงข้อมูลล่าสุดจาก Supabase ทันที ถ้ายังไม่เปลี่ยน — ตรวจสอบว่า Supabase project ไม่ได้ "Paused"
Supabase ขึ้น "Paused" หรือ "Unhealthy" ทำอย่างไร?
เข้า supabase.com → คลิกโปรเจกต์ → กดปุ่ม Resume project สีเขียว → รอ 1–2 นาที → Refresh หน้าแอป → ใช้งานได้ปกติ
เกิดจาก Free tier หยุดทำงานอัตโนมัติเมื่อไม่ใช้งาน 7 วัน
อยากเพิ่มเมนูใหม่ แต่ยังไม่มีรูป ทำได้ไหม?
ได้ครับ ระบบจะแสดงรูป placeholder แทน แต่แนะนำให้ส่งรูปอาหารจริงมาให้เราใส่ให้ (นับเป็นการแก้ไข 1 ครั้ง) เพราะรูปสวยช่วยให้ลูกค้าตัดสินใจสั่งได้ดีกว่ามาก
แก้ชื่อเมนูหรือคำอธิบายเองได้ไหม (Standard / Premium)?
ได้ครับ ทำเหมือนแก้ราคา — Table Editor → คลิกเซลล์ name (ชื่อ) หรือ description (คำอธิบาย) → แก้ข้อความ → Enter บันทึกทันที
ออเดอร์ไม่เข้า LINE OA (แพ็ก Premium)?
ตรวจสอบว่า LINE OA ยังออนไลน์ และ Token ยังใช้งานได้ ถ้ายังผิดปกติหลังตรวจสอบแล้ว → แจ้งเราได้เลยทาง LINE เราตอบกลับภายใน 24 ชั่วโมง
อยากเปลี่ยนสีหรือโลโก้ร้านหลัง deploy แล้ว ทำได้ไหม?
ได้ครับ ส่งสีใหม่ (hex code เช่น #FF5733) และโลโก้ใหม่มาให้เรา — นับเป็นการแก้ไข 1 ครั้ง (ใช้สิทธิ์แก้ไขฟรีได้)
ต้องการ URL เป็นชื่อร้านเอง เช่น menu.myshop.com ทำได้ไหม?
ได้ครับ ต้องซื้อ domain ก่อน (ประมาณ 500–800 บาท/ปี) แล้วแจ้งเรา — เราจะตั้งค่าให้ มีค่าบริการเพิ่มเติม ทักมาสอบถามได้เลย
อยากเพิ่มหมวดหมู่ใหม่ (เช่น "เมนูพิเศษ") ทำเองได้ไหม?
ส่วนนี้ต้องให้เราทำให้ครับ เพราะต้องแก้ในโค้ดแอป — แจ้งชื่อหมวดใหม่มาได้เลย นับเป็นการแก้ไข 1 ครั้ง
ใช้สิทธิ์แก้ไขฟรีหมดแล้ว ครั้งต่อไปเสียค่าใช้จ่ายเท่าไร?
คิดตามขอบเขตงาน:
· แก้ข้อความ / ราคา → 150฿/ครั้ง
· เพิ่มเมนูพร้อมรูป → 200฿/รายการ
· แก้ดีไซน์ใหญ่ (สี / โลโก้ / เลย์เอาต์) → เริ่มต้น 500฿
ทักมาได้เลย เราจะ quote ราคาให้ก่อนเสมอ

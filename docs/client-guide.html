🍽️ ระบบเมนู QR Code คืออะไร?
บทนำและภาพรวมการใช้งาน

ระบบเมนู QR Code คือ เมนูอาหารออนไลน์ ที่ลูกค้าร้านสแกน QR Code แล้วดูเมนูและสั่งอาหารได้ทันทีบนมือถือ โดยไม่ต้องดาวน์โหลดแอปใด ๆ

1
🖨️ พิมพ์ QR Code ติดที่โต๊ะ ได้รับไฟล์ QR Code จากเรา → พิมพ์ติดโต๊ะ / เคาน์เตอร์ / บอร์ดหน้าร้าน
2
📱 ลูกค้าสแกนด้วยมือถือ เปิดกล้องหรือแอปสแกน QR → ระบบเปิดเมนูในเบราว์เซอร์ทันที ไม่ต้องดาวน์โหลด
3
🛒 เลือกเมนูและสั่งอาหาร ดูรูป เลือกตัวเลือก (ขนาด/เนื้อ/ความหวาน) เพิ่มลงตะกร้า แล้วยืนยัน
4
💬 ออเดอร์เข้า LINE ทันที (แพ็ก 3) ร้านได้รับออเดอร์พร้อมรายการ ราคา โต๊ะ ผ่าน LINE OA โดยอัตโนมัติ
🌟 ข้อดีเทียบกับเมนูกระดาษ
  • แก้ราคาได้ทันที ไม่ต้องพิมพ์ใหม่
  • ซ่อนเมนูที่หมดได้ชั่วคราว
  • ลูกค้าดูรูปอาหารสวยงาม ช่วยตัดสินใจ
  • รองรับ 2 ภาษา ไทย/อังกฤษ (แพ็ก 2–3)
  • สั่งผ่าน LINE ลดความผิดพลาด (แพ็ก 3)

  • ชื่อร้าน
  • รายการเมนูสูงสุด 20 รายการ (ชื่อ + ราคา + หมวดหมู่)

การใช้งาน

1
พิมพ์ QR Code รับไฟล์ PNG จากเรา → พิมพ์ใส่กระดาษ/อคริลิค/สติกเกอร์ ติดที่โต๊ะหรือเคาน์เตอร์
2
ลูกค้าสแกน QR เปิดกล้องมือถือ → ชี้ที่ QR Code → กดลิงก์ที่ขึ้นมา → เมนูเปิดในเบราว์เซอร์ทันที
3
ลูกค้าดูเมนูและแจ้งพนักงาน ดูรูป ราคา เลือกเมนูที่ต้องการ → แจ้งพนักงานสั่ง (แพ็กนี้ยังไม่รองรับสั่งผ่านระบบ)
⚠️ ข้อจำกัดของแพ็ก Basic ถ้าต้องการแก้ราคาหรือเพิ่มเมนูในภายหลัง จะต้องส่งข้อมูลใหม่ให้เราแก้ (ใช้สิทธิ์แก้ไขฟรี 1 ครั้ง หรือคิดเพิ่ม) — แนะนำ อัปเกรดแพ็ก Standard เพื่อแก้ราคาเองได้ไม่จำกัด

Flow การสั่งอาหาร (Dine In)

1
ลูกค้าสแกน QR ที่โต๊ะ → เลือกเมนู ดูรูป อ่านคำอธิบาย เลือกตัวเลือก (เนื้อ/ขนาด/ความหวาน) → เพิ่มลงตะกร้า
2
ตรวจรายการในตะกร้า → กดยืนยัน ระบุโต๊ะ → เลือกวิธีชำระเงิน (QR PromptPay / เงินสด)
3
ออเดอร์ส่งเข้า LINE OA ร้านทันที ร้านได้รับข้อความ: โต๊ะ X · รายการ · ราคา · วิธีชำระ ครบครัน
4
ลูกค้าสแกน QR จ่ายเงิน ถ้าเลือก QR PromptPay → แนบสลิปส่งกลับ LINE OA

Flow การสั่ง Delivery

1
ลูกค้ากด "ส่งถึงบ้าน" เลือกเมนู → กรอกชื่อ เบอร์โทร ที่อยู่ เลือกรอบส่ง (เช้า/บ่าย)
2
ยืนยันออเดอร์ → สแกนชำระเงิน ระบบแสดง QR PromptPay → ลูกค้าโอนแนบสลิป + ส่งโลเคชั่น
3
ร้านได้รับออเดอร์ + ที่อยู่ครบใน LINE OA ยืนยัน → จัดส่งตามรอบที่ลูกค้าเลือก

🔖 แก้ราคาเมนู

1
เปิด supabase.com → Sign in ใช้ email/password ที่ลงทะเบียนไว้
2
คลิกโปรเจกต์ร้านของคุณ หน้า Dashboard จะขึ้นมา
3
เมนูซ้ายมือ → Table Editor → menu จะเห็นตารางเมนูทั้งหมด คอลัมน์ name / price / cat ฯลฯ
4
คลิกเซลล์ price ของเมนูที่ต้องการ → พิมพ์ราคาใหม่ → Enter ระบบบันทึกทันที เมนูในแอปอัปเดตภายใน 1-2 วินาที

➕ เพิ่มเมนูใหม่

1
Table Editor → menu → กดปุ่ม Insert (สีเขียว) หน้าต่าง "New row" เปิดขึ้น
2
กรอกข้อมูลเมนูใหม่ id (เลขไม่ซ้ำ), cat (หมวด), name (ชื่อ), price (ราคา), description (คำอธิบาย)
3
กด Save → เมนูขึ้นในแอปทันที หมายเหตุ: รูปภาพต้องให้เราอัปเดตให้ (ใช้สิทธิ์แก้ไขฟรี)

🔧 แก้ด้วย SQL (เร็วกว่า — สำหรับแก้หลายรายการ)

ถ้าต้องการแก้ราคาหลายเมนูพร้อมกัน ใช้ SQL Editor จะเร็วกว่ามาก

-- แก้ราคาเมนูเดียว (id คือหมายเลขเมนู)
UPDATE public.menu SET price = 80 WHERE id = 4;

-- แก้ราคาหลายเมนูพร้อมกัน
UPDATE public.menu SET price = 70 WHERE id IN (4, 5, 6);
⚠️ ถ้า Supabase ขึ้น "Unhealthy" หรือ "Paused" Supabase Free tier จะ pause อัตโนมัติเมื่อไม่มีการใช้งาน 1 สัปดาห์ → กดปุ่ม Resume → รอ 1-2 นาที → ใช้งานได้ปกติ
ถ้าร้านเปิดทุกวัน แนะนำอัปเกรด Supabase Pro ($25/เดือน) เพื่อไม่ให้ pause
👁️ ซ่อน / แสดงเมนูชั่วคราว
สำหรับแพ็ก Standard และ Premium

ถ้าเมนูหมดชั่วคราว หรืออยากซ่อนไว้ก่อน — ทำได้โดยไม่ต้องลบเมนูออก

1
Table Editor → ตาราง menu หาเมนูที่ต้องการซ่อน
2
คลิกเซลล์ is_available ค่าปัจจุบัน = true (แสดง) → เปลี่ยนเป็น false (ซ่อน)
3
กด Enter → เมนูหายจากแอปทันที เมื่อต้องการแสดงอีกครั้ง → เปลี่ยนกลับเป็น true
💡 วิธีด่วนผ่าน SQL
-- ซ่อนเมนู
UPDATE public.menu SET is_available = false WHERE id = 5;

-- แสดงเมนูอีกครั้ง
UPDATE public.menu SET is_available = true WHERE id = 5;
❓ คำถามที่พบบ่อย (FAQ)
ปัญหาที่ลูกค้ามักเจอ และวิธีแก้
Q: ลูกค้าสแกน QR Code แล้วไม่เปิด?
ลองใช้กล้องมือถือปกติ (ไม่ต้องโหลดแอปพิเศษ) ถ้ายังไม่ขึ้น ลองกด QR Code ใน Gallery แล้ว "เปิดลิงก์" หรือใช้แอป LINE สแกนก็ได้
Q: แก้ราคาใน Supabase แล้วแต่เมนูในแอปยังเป็นราคาเดิม?
โหลดหน้าใหม่ (ดึงหน้าจอลง/Refresh) ระบบจะดึงข้อมูลใหม่จาก Supabase ทันที ถ้ายังไม่เปลี่ยน ตรวจสอบว่า Supabase project ไม่ได้ "Paused"
Q: Supabase ขึ้น "Paused" / "Unhealthy" ทำอย่างไร?
เข้า supabase.com → คลิกโปรเจกต์ → กดปุ่ม Resume project สีเขียว → รอ 1-2 นาที → Refresh หน้า → ใช้งานได้ปกติ (เกิดจาก Free tier pause อัตโนมัติเมื่อไม่ใช้งาน 7 วัน)
Q: อยากเพิ่มเมนูใหม่ แต่ไม่มีรูป ทำได้ไหม?
ได้ครับ ระบบจะแสดงรูป placeholder แทน แต่แนะนำส่งรูปอาหารจริงมาให้เราใส่ให้ (ใช้สิทธิ์แก้ไขฟรี) เพราะรูปสวยช่วยให้ลูกค้าตัดสินใจสั่งได้ดีกว่า
Q: อยากเปลี่ยนชื่อเมนูหรือคำอธิบาย ทำเองได้ไหม?
ได้ครับ ทำเหมือนแก้ราคา — Table Editor → คลิกเซลล์ name หรือ description → แก้ข้อความ → Enter บันทึกทันที
Q: ออเดอร์ไม่เข้า LINE OA (แพ็ก 3)?
ตรวจสอบว่า LINE OA ยังออนไลน์อยู่ และ LINE Channel Access Token ยังใช้งานได้ ถ้ายังผิดปกติ แจ้งเราได้เลย
Q: อยากเปลี่ยนสีหรือโลโก้ร้านหลัง deploy แล้ว ทำได้ไหม?
ได้ครับ ส่งสีใหม่ (hex code) และโลโก้ใหม่มาให้เรา — นับเป็นการแก้ไข 1 ครั้ง (ใช้สิทธิ์แก้ไขฟรีได้)
Q: ต้องการ URL เป็นชื่อร้านเอง เช่น menu.ชื่อร้าน.com ทำได้ไหม?
ได้ครับ ต้องซื้อ domain ก่อน (ประมาณ 500-800 บาท/ปี) แล้วแจ้งเรา เราจะตั้งค่าให้ มีค่าบริการเพิ่มเติม

ออเดอร์เข้า ]
วิธีเข้า Supabase 🗄️
🌐 supabase.com
  • 1
    เปิดเบราว์เซอร์ → ไปที่ supabase.com → กด Sign In
  • 2
    Login ด้วย email + password ที่เราส่งให้
  • 3
    กดเข้า Project ของร้าน (ชื่อเหมือนชื่อร้าน)
  • 4
    ที่เมนูซ้าย กด Table Editor → เลือก menu_items
วิธีแก้ราคาเมนู ✏️
  • ใน Table Editor เลือก table menu_items
  • ดับเบิลคลิก ที่ช่อง price ของเมนูที่ต้องการแก้
  • พิมพ์ราคาใหม่ → กด Enter หรือ Save Changes
  • เปิดหน้าเมนูของร้านใหม่ → ราคา อัปเดตทันที!
-- ตัวอย่าง SQL (ไม่บังคับใช้ ถ้า UI ง่ายกว่า) UPDATE menu_items SET price = 65 WHERE name_th = 'ข้าวหมูแดง';
วิธีซ่อน / แสดงเมนู 👁️
  • ทุกรายการเมนูมีช่อง is_available (true = แสดง, false = ซ่อน)
  • คลิกที่ช่อง is_available → สลับเป็น false → เมนูหายจากแอปทันที
  • เปลี่ยนกลับเป็น true เมื่อสินค้ากลับมา → เมนูโผล่ขึ้นมาเอง
💡 ใช้เมื่อ
วัตถุดิบหมด · เมนูตามฤดูกาล · ปิดชั่วคราว
is_available: true
is_available: false
วิธีเพิ่มเมนูใหม่ ➕
  • 1
    ใน Table Editor กด Insert → Insert Row
  • 2
    กรอก field ที่จำเป็น: name_th, name_en, price, category_id
  • 3
    ตั้ง is_available = true เพื่อให้แสดงในแอปทันที
  • 4
    กด Save → เมนูใหม่ปรากฏในแอปทันที ✓
-- Fields ที่ต้องกรอก name_th = 'ชื่อเมนูภาษาไทย' name_en = 'Menu Name in English' price = 80 category_id = 1 -- ดู ID จาก table categories is_available = true
สลับภาษา ไทย / EN 🌐
  • ในหน้าเมนูมี ปุ่มสลับภาษา อยู่มุมขวาบน (TH / EN)
  • กด EN → ชื่อเมนู คำอธิบาย หมวดหมู่ เปลี่ยนเป็นอังกฤษทันที
  • ลูกค้าเลือกภาษาเองได้ตามใจ · ไม่กระทบออเดอร์ที่ส่ง LINE

QR Code โต๊ะ + Delivery 🪑
Dine-in — QR แยกโต๊ะ
  • เราส่ง QR Code PNG ทุกโต๊ะ ให้แล้ว (สูงสุด 30 โต๊ะ)
  • ปริ้นท์ใส่กรอบหรือแลมิเนต วางบนโต๊ะได้เลย
  • แต่ละ QR ระบุหมายเลขโต๊ะอัตโนมัติ เช่น โต๊ะ 1, 2, 3...
Delivery — สั่งส่งบ้าน
  • ลูกค้าเข้าผ่าน ลิงก์เมนู ที่ร้านแชร์
  • กรอก ชื่อ เบอร์ ที่อยู่ → ออเดอร์เข้า LINE OA
  • ไม่ต้องโทรหากัน ข้อมูลครบในข้อความเดียว

แพ็ก Standard รองรับ 2 ภาษา ถ้ามีชื่อเมนูภาษาอังกฤษอยู่แล้วก็ยิ่งดี ถ้าไม่มีเราจะแปลให้ครับ

📋 Checklist ข้อมูลที่ต้องส่ง

  • ชื่อร้าน (ไทย + อังกฤษถ้ามี) เช่น ร้านข้าวมูหมู / Khaomoo Restaurant
  • รายการเมนูสูงสุด 60 รายการ ชื่อไทย + ราคา + หมวดหมู่ · ไม่มีชื่ออังกฤษก็ได้ เราจะแปลให้
  • รูปอาหารทุกรายการ (ถ้ามี) ยิ่งครบยิ่งดี ถ่ายจากมือถือแสงสว่างได้เลย
  • โลโก้ร้าน (PNG หรือ JPG) โลโก้จะแสดงในหัวเมนู
  • สีหลักและสีรองของร้าน เช่น "แดงเลือดหมู" หรือ hex code เช่น #8B2635
  • อีเมลสำหรับสร้างบัญชี Supabase ใช้บัญชีนี้เข้าแก้ราคาเองในภายหลัง เราจะส่งคำเชิญให้
💡 Supabase คืออะไร? Supabase คือเว็บสำหรับจัดการข้อมูลเมนูร้านของคุณ เหมือน Google Sheets แต่เชื่อมกับแอปเมนูโดยตรง แก้ราคาในนั้น → เมนูในแอปอัปเดตทันที — ใช้ผ่านเบราว์เซอร์ปกติ ไม่ต้องติดตั้งโปรแกรม

🔖 วิธีแก้ราคาเมนู

1
เปิดเบราว์เซอร์ → พิมพ์ supabase.com → กด Sign in

กรอก email + password ที่เราส่งให้ กด Log in

2
เลือกโปรเจกต์ร้านของคุณ

จะเห็นชื่อโปรเจกต์ (ชื่อร้าน) อยู่หน้าแรก กดคลิกเข้าไป

3
เมนูซ้ายมือ → กด "Table Editor"

จะเห็นรายการตาราง → คลิกตาราง menu

4
หาเมนูที่ต้องการแก้ → คลิกเซลล์ "price"

คลิกที่ตัวเลขราคา → พิมพ์ราคาใหม่ → กด Enter → บันทึกทันที

✅ อัปเดตทันทีเลยครับ! กด Enter แล้ว → แอปเมนูอัปเดตราคาใหม่ภายใน 1–2 วินาที ลูกค้าโหลดหน้าใหม่จะเห็นราคาใหม่ทันที ไม่ต้อง deploy ใหม่ ไม่ต้องแจ้งเรา

➕ วิธีเพิ่มเมนูใหม่

1
Table Editor → menu → กดปุ่ม "Insert row" (สีเขียว)

หน้าต่าง "New row" เปิดขึ้น

2
กรอกข้อมูลเมนูใหม่

id (เลขไม่ซ้ำ) · cat (หมวด) · name (ชื่อ) · price (ราคา) · description (คำอธิบาย)

3
กด Save → เมนูขึ้นในแอปทันที

หมายเหตุ: รูปภาพต้องให้เราใส่ให้ (นับเป็น 1 ครั้งของสิทธิ์แก้ไขฟรี)

⚡ แก้หลายรายการพร้อมกันด้วย SQL (สำหรับคนที่อยากเร็วขึ้น)

ถ้าต้องการแก้ราคาหลายเมนูพร้อมกัน → เมนูซ้าย → SQL Editor → New query → วางโค้ดนี้ → กด Run

-- แก้ราคาเมนูเดียว (id คือหมายเลขเมนู เช่น 4 = ข้าวขาหมู) UPDATE public.menu SET price = 80 WHERE id = 4; -- แก้ราคาหลายเมนูพร้อมกัน UPDATE public.menu SET price = 70 WHERE id IN (4, 5, 6); -- แก้ชื่อเมนู UPDATE public.menu SET name = 'ขาหมูพะโล้สูตรใหม่' WHERE id = 1;
⚠️ ถ้า Supabase ขึ้น "Paused" Supabase แบบฟรีจะหยุดอัตโนมัติถ้าไม่ใช้งาน 7 วัน
วิธีแก้: เข้า supabase.com → คลิกโปรเจกต์ → กด Resume project → รอ 1–2 นาที → ใช้งานได้ปกติ

👁️ วิธีซ่อนเมนูชั่วคราว

เมื่อวัตถุดิบหมด หรืออยากปิดเมนูบางอย่างในวันนั้น → ทำได้ง่ายมากครับ ไม่ต้องลบ กลับมาเปิดเมื่อไรก็ได้

1
Supabase → Table Editor → ตาราง menu

หาเมนูที่ต้องการซ่อน

2
มองหาคอลัมน์ "is_available"

ค่าปัจจุบัน = true (แสดงอยู่)

3
คลิกเซลล์ → เปลี่ยนเป็น false → กด Enter

เมนูหายจากแอปทันที ลูกค้าจะไม่เห็น

4
อยากเปิดอีกครั้ง → เปลี่ยนกลับเป็น true → Enter

เมนูกลับมาแสดงทันที

✅ ไม่ต้องกลัวข้อมูลหาย การซ่อน (false) ไม่ได้ลบข้อมูลออก แค่ซ่อนจากหน้าแอปเท่านั้น เปลี่ยนกลับ true ได้ตลอดเวลา

❓ คำถามที่พบบ่อย

แก้ราคาแล้วแต่ยังเห็นราคาเดิม?
โหลดหน้าใหม่ (ดึงหน้าจอลง / กด Refresh) ระบบจะดึงข้อมูลใหม่ทันที ถ้ายังไม่เปลี่ยน → ตรวจว่า Supabase ไม่ได้ Paused
Supabase ขึ้น "Paused" ทำอย่างไร?
เข้า supabase.com → คลิกโปรเจกต์ → กดปุ่ม Resume project สีเขียว → รอ 1–2 นาที → Refresh หน้าแอป → ใช้งานได้ปกติ (เกิดจาก Free tier หยุดอัตโนมัติเมื่อไม่ใช้ 7 วัน)
อยากเพิ่มหมวดหมู่ใหม่ ทำเองได้ไหม?
ส่วนนี้ต้องให้เราทำให้ครับ เพราะต้องแก้ในโค้ดแอป แจ้งชื่อหมวดใหม่มาได้เลย นับเป็นการแก้ไข 1 ครั้ง
อยากเปลี่ยนสีหรือโลโก้ ทำได้ไหม?
ได้ครับ ส่งสีใหม่ (hex code) และโลโก้ใหม่ให้เรา นับเป็นการแก้ไข 1 ครั้ง (ใช้สิทธิ์แก้ไขฟรีได้)
ใช้สิทธิ์แก้ไขฟรีหมดแล้ว ครั้งต่อไปเสียเท่าไร?
แก้ข้อความ / ราคา → 150฿/ครั้ง · เพิ่มเมนูพร้อมรูป → 200฿/รายการ · แก้ดีไซน์ → เริ่มต้น 500฿
ทักมาได้เลย เราจะ quote ให้ก่อนเสมอ
